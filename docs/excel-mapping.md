对照文件：`001679511(3).xlsx`（与本文档同目录；年度更新対象年度：2026 / 令和8年）

## 工作表用途

| 工作表 | 用途 | Web 对应 |
| --- | --- | --- |
| 利用方法・注意事項 | 使用说明与注意事项 | 页面提示、校验提示 |
| 算定基礎賃金集計表 | 12个月、3次赏与、人数及工资的主要输入表 | 事業情報、賃金集計 |

## 字段映射

已覆盖：

- 事業名称、所在地、邮编、电话、劳动保险编号
- 具体业务或作业内容
- 出向者受入人数、送出人数
- 12个月的常用、役员、临时劳动者以及雇用保险对象人数/工资
- 3次赏与的同类人数/工资
- 最多5名“役員で労働者扱い”明细：姓名、职位、雇用保险资格
- 劳灾、雇用（确定/概算）、一般拠出金费率
- 已申告概算保险费、纳付次数、充当/还付方式与充当目标

原表中的劳动保险编号分格输入在 Web 中合并为一个字符串保存；导出时保留完整号码。

## 公式映射

- 劳灾、雇用及一般拠出金的算定基础额：全年工资（含赏与）舍去不足1,000円。
- 劳灾与雇用的算定基础额相同时，按合并费率计算后舍去不足1円；不相同时分别计算后相加。
- 一般拠出金按劳灾算定基础额计算并舍去不足1円。
- 一元适用的3回分纳门槛为概算保险费400,000円；二元适用为200,000円。
- 分纳时概算保险费除以3，不能整除的余数加入第1期。
- 一般拠出金不能分纳，全额加入第1期。
- 已申告概算保险费高于确定保险费时，可选择优先充当后还付余额，或全额还付；充当目标可选劳动保险费、一般拠出金或两者。

## 边界与上线前确认

- 原 Excel 的“申告書記入イメージ”明确注明不能打印后作为正式申告书提交；当前 PDF/Excel 同样定位为计算及转记辅助材料。
- 劳灾保险率随具体业务种类变化，不能仅凭“普通/农林水产/建设”自动确定，必须由用户对照官方费率表输入。
- 事业成立、废止日期、废止理由、片保险理由代码等属于正式申告书/e-Gov 转记字段，目前未参与工资集计与保险费计算。若后续要生成完整转记清单或 e-Gov 数据，需要追加这些字段。
- 建设项目、海外派遣者、特别加入者等特殊计算不应套用本通用模型，需要另建计算流程。
